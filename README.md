# Igo Browser Game

ブラウザだけで動く、人対人の囲碁ゲームです。

## Features

- 9路盤、13路盤、19路盤の切り替え
- 各盤面の星表示
- 開始時の置き石設定
- 黒白交互の対人対局
- 石取り、呼吸点判定、自殺手禁止、単純コウ禁止
- パス、投了、手動終局、リセット
- 手番、手数、アゲハマ表示
- 中国式エリア計算による途中のザックリ試算と終局計算

## Play

GitHub Pagesで公開された `index.html` を開くだけで動きます。外部CSS、外部JavaScript、CDNには依存していません。

## Scope

試算と終局計算は中国式エリア計算の簡易版です。盤上の石と単色で囲まれた空点を数えます。死石指定はありません。
